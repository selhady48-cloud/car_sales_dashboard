# car_sales_dashboard-final
An interactive web dashboard used to explore used car listings data in the U.S.

This application allows users to visually analyze used vehicle prices compared to the model year and mileage. (Built with Streamlit)

# Features
- Interactive controls to adjust outliers for clearier information
- Histogram displaying the distribution of used car prices
- Scatter plot displaying the distribution of used car prices to odometer (mileage)
- Scatter plot displaying used car prices to the model year

# Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
- Docker
- Render
